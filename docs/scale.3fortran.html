NAME

SCALE(3) - [MODEL_COMPONENTS] Scale a real value by a whole power of the radix

SYNOPSIS

result = scale(x, i) 

         elemental real(kind=KIND) function scale(x, i)


          real(kind=KIND),intent(in) :: x
          integer,intent(in)         :: i

CHARACTERISTICS

DESCRIPTION

SCALE(3) returns x * RADIX(X)**I.

OPTIONS

o X : The type of the argument shall be a real.
o I : The type of the argument shall be a integer.

RESULT

The return value is of the same type and kind as X. Its value is X * RADIX(X)**I.

EXAMPLES

Sample program: 

    program demo_scale
    implicit none
    real :: x = 178.1387e-4
    integer :: i = 5
       print *, scale(x,i), x*radix(x)**i
    end program demo_scale
Results: 

        0.570043862      0.570043862

STANDARD

Fortran 95
